# NeoArt
Nền tảng chia sẻ nghệ thuật trực tuyến — nơi người dùng khám phá, đăng tải và tương tác với các tác phẩm sáng tạo.

# Tính năng
- Đăng tải và xem bài viết có nhiều ảnh
- Thích, bình luận, trả lời bình luận
- Lưu bài vào thư mục cá nhân
- Thông báo realtime (WebSocket)
- Tìm kiếm bài viết theo từ khóa
- Trang hồ sơ cá nhân
- Nhắn tin trực tiếp

# Cách cài đặt
1. Cài clone dự án 
- git clone https://github.com/KimThuong1510/neoart.git
- cd neoart
2. Cấu hình Database trong application.properties
- spring.datasource.url=jdbc:mysql://localhost:3306/neoart
- spring.datasource.username=root
- spring.datasource.password=yourpassword
3. Chạy ứng dụng
mvn spring-boot:run

**Truy cập tại**: http://localhost:8080
